Projects

Dropmap prediction tool

A personal project where I developed a machine learning tool that predicts optimal drop locations in a game map based on historical player data. While originally inspired by game scenarios, the project focuses on geospatial data analysis and AI-based decision-making.

I built a convolutional neural network (CNN) using TensorFlow and Keras to predict multiple (x, y) coordinates on a map image. The model was trained on custom image data paired with manually annotated locations stored in JSON format. Images were preprocessed and normalized, and predictions were visualized by overlaying the output coordinates on resized map images.

Custom Discord Bot

A lightweight yet versatile Discord bot built with Python using the discord.py library. This bot was developed to enhance engagement and moderation capabilities in a private community server.

Key features include moderation commands, along with dynamic role assignment featuring color customization. It also includes utility tools like real-time weather lookup using the OpenWeatherMap API. Additionally, the bot supports fun interactions including image and GIF responses, personalized command sets, and randomized joke replies.
